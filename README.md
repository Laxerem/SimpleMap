# SimpleMap

> [!FAQ]
> This project has been published, and you can see it there: https://laxerem.github.io/lastland.github.io/home

## How to build

### 1. Clone this repository

```bash
git clone https://github.com/Laxerem/SimpleMap.git
```

### 2. Intstall dependencies

Go to the project folder (frontend), and execute this command:

```bash
npm install
```

### 3. Launch project

```bash
npm run dev
```

After that, you can see the website in browser by clicking on the link that you will see in the console.
